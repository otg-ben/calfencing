# California Fence Company — Cal Fence Website

## Project Overview
Static marketing website for California Fence Company (calfencing.com), owned by Bob Easley. Rebuilding from an old Wix site. Static export — no backend, no CMS needed.

## Tech Stack
- Next.js 14 App Router, static export (`output: 'export'`)
- TypeScript
- Tailwind CSS
- Google Fonts: Playfair Display (headings) + Inter (body)

## URL Structure (DO NOT CHANGE — preserving for SEO/AdWords)
- `/` → Home
- `/about` → About
- `/products` → Products
- `/gallery` → Gallery
- `/untitled` → Contact (yes, this slug is intentionally ugly — leave it)
